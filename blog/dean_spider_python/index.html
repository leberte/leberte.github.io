一、摘要

最近在学习python的爬虫，突然想到一个点子：学校教务以及学生工作处每天都在更新信息，如果每天都去访问网站的话显得太浪费时间，但是有些信息又非常重要，那么可不可以有一个程序直接把消息发给我呢？于是，诞生了这个项目。
实现这个项目，首先，我们需要将学校教务网上的信息爬取出来，然后进行筛选，将最新的消息通过email的形式发送到我的邮件。最近浏览GitHub时发现大佬写过一个QQ机器人的项目，想到我可以更加直接的推送到常常使用的QQ上，就诞生后面的QQ通知。
在实现数据处理时，mysql小白的我经历过痛苦的事情，具体问题后面解释。

3、定时服务的配置

在Linux系统中存在crontab的服务，也就是定时服务。

3.1 定时服务字符的解释

在crontab文件中，表头文件就如下面所示，#就不用看，主要看下面对参数的解释。

  • minute - 从0到59的整数
  • hour - 从0到23的整数
  • day - 从1到31的整数 (必须是指定月份的有效日期)
  • month - 从1到12的整数 (或如Jan或Feb简写的月份)
  • dayofweek - 从0到7的整数，0或7用来描述周日 (或用Sun或Mon简写来表示)
  • command - 需要执行的命令(可用as ls /proc >> /tmp/proc或 执行自定义脚本的命令)
